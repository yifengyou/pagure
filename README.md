## pagure项目简介

![20210713_132216_59](image/20210713_132216_59.png)


* Pagure 这个名字取自法语单词“pagere”。法语中的 Pagure 被用作Paguroidea 超科甲壳类动物的通用名称，该超科基本上是寄居蟹的科。
* Paugre 是一个典型的git Server，类似于gitlab、github
* 基于gitolite、pygit2
* fedora、centos开源软件托管仓库
